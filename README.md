Celebrity Prediction Market Tracker — find asymmetric opportunities before the market catches on.

## Deploy

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/ace28888888/polymarket-tracker)

Or manual:
1. Fork/clone this repo
2. Connect to [Render](https://render.com)
3. Use Python runtime, `pip install -r requirements.txt`, `streamlit run app.py --server.port $PORT`
